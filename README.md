# ChordsWebApp
 Web Application to automatic musical Chords and Lyrics Transcription
 ---
 
 ## Enviroment
Python 3.8.5

```bash
pip install -r requirements.txt
```

## Running

```bash
flask run #runs app
```
Open in a Web browser
Website: http://localhost:5000
